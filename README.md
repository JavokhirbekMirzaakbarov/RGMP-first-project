# React JS Global Mentoring Program First Project

This is the first project from React Fundamentals mentoring program at EPAM

## Features Available

- Login/Logout
- Add/Update/Delete todos
- Show/Sort customers list

### Technologies

- React/ReactDOM/React-router
- ChakraUI
- GithubAPI
- Formik
